# Sistema de Inspeção Bimestral de Segurança - DASCO Engenharia

Sistema completo para inspeção de segurança em veículos com geração de PDF e envio via WhatsApp.

## Funcionalidades

✅ Formulário completo com 19 itens de inspeção
✅ Salvamento automático em cache (localStorage)
✅ Geração de PDF profissional
✅ Envio via WhatsApp
✅ PWA (Progressive Web App) - pode ser instalado no celular
✅ Design responsivo para mobile e desktop
✅ Tudo centralizado e em uma única página A4

## Como Usar

1. Abra o arquivo `index.html` no navegador
2. Preencha todos os campos do formulário
3. Os dados são salvos automaticamente no cache
4. Gere o PDF ou envie via WhatsApp

## PWA - Instalação no Celular

Para instalar como aplicativo no celular:

1. Abra o sistema no navegador do celular
2. No Chrome/Edge: Menu → "Adicionar à tela inicial"
3. No Safari: Compartilhar → "Adicionar à Tela de Início"

## Ícones PWA

Para funcionar completamente como PWA, você precisa criar dois ícones:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

Você pode criar esses ícones usando qualquer editor de imagens ou ferramenta online.

## Cache

O sistema salva automaticamente todos os dados preenchidos. Mesmo fechando o navegador, ao abrir novamente, tudo estará preenchido.

Para limpar o cache, clique no botão "Limpar Cache".

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- jsPDF (geração de PDF)
- Service Worker (PWA)
- LocalStorage (cache)
